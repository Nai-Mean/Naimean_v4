## 3. Added global CSS reset
Standardized the base styles to ensure consistency across different browsers and components.

### Why
- Eliminates browser-specific default margins and paddings
- Provides a clean slate for custom styling
- Fixes layout inconsistencies in the Aquarium and Den views

### What Was Added
A new `reset.css` file was included in the global styles.

### How to Use
- This is automatically applied globally.
- No further action required by developers.
